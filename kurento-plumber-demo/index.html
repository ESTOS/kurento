Kurento Browser JavaScript: Plumber element demo

This application shows a WebRtcEndpoint connected to a HttGetEndpoint instantiated in other pipeline. Both pipelines will be connected using a PlumberEndpoint that will send media capabilities and other media events back and forward through the internal SCTP connection so that negotiation of capabilities can succeed in a transparent way before the media start flowing between both pipelines. Take a look to the Media Pipeline. To run this demo follow these steps:

  1. Open this page with a browser compliant with WebRTC (Chrome, Firefox).
  2. Click on Start button.
  3. Grant the access to the camera and microphone. After the SDP negotiation the HttpGetEndpoint should start showing media.
  4. Click on Stop to finish the communication.
